# Student Management System (C++)

## Description
A console-based C++ application that allows users to manage student records.

## Features
- Add student records
- Display all students
- Store names and marks

## Concepts Used
- Structures
- Vectors
- Functions
- Loops

##  How to Run
1. Compile using a C++ compiler (e.g., g++)
2. Run the program in terminal
